# Hok Status 🐔

Simpele app die laat zien of het hok open of dicht is.

- **Groen** → Hok is open
- **Rood** → Hok is dicht
- **Grijs** → Laden...

## Features

- Automatische refresh elke 30 seconden (met countdown)
- Pull-to-refresh
- Trilsignaal (~5s) wanneer het hok opengaat
- Toont tijdstip van laatste statuswijziging
